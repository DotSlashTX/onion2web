Onion2web

Access .onion sites without Tor Browser

Download this project as a .zip file Download this project as a tar.gz file

This site is reverse proxy for Tor's onion sites.

To access onion sites using onion.gq, append .gq to its name. Example: access onion search engine "not Evil" hss3uro2hsxfogfq.onion as hss3uro2hsxfogfq.onion.gq.

Another example: listen Web Radio: http://76qugh5bey5gum7l.onion.gq

This machine does not host or contain any illegal content. Network infrastructure maintainers are not liable for the type of content that passes over their equipment.
